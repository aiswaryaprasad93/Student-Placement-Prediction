# Student-Placement-Prediction

## Overview
This project predicts whether a student is likely to be placed based on academic performance using a Machine Learning model.

## Features
- Predicts student placement status
- User-friendly Streamlit web application
- Takes SSC, HSC, Degree, and MBA percentages as input
- Displays placement result as Placed or Not Placed

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Machine Learning Algorithm
- Logistic Regression

## Accuracy
- 88.37%

## Dataset
- Placement_Data_Full_Class.csv
